# 🚀 Sham Cash Project

## 📌 وصف المشروع
هذا موقع "شام كاش" مع لوحة تحكم (Admin Dashboard) لعرض بيانات الزوار وإدارتهم.

## ⚙️ المميزات
- تسجيل دخول المستخدمين
- عرض بيانات الزوار في لوحة التحكم
- قبول / رفض الطلبات
- ربط مع Firebase
- تحديث مباشر للبيانات (Realtime)

## 🖥️ التقنيات المستخدمة
- Next.js
- Firebase (Database)
- Tailwind CSS
- Vercel (Deployment)
