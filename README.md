# 🚀 Sham Cash Project

## 📌 وصف المشروع
هذا موقع "شام كاش" مع لوحة تحكم (Admin Dashboard) لعرض بيانات الزوار وإدارتهم.

## ⚙️ المميزات
- تسجيل دخول المستخدمين
- عرض بيانات الزوار في لوحة التحكم
- قبول / رفض الطلبات
- ربط حي بين الموقع ولوحة التحكم عبر API
- تحديث شبه لحظي للبيانات (Polling + Server Store)

## 🖥️ التقنيات المستخدمة
- Next.js
- Next.js API Routes
- Tailwind CSS
- Vercel (Deployment)
